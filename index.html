System Overview

This system implements a novel two-phase hybrid deep learning approach for glioma tumor detection and grade classification, achieving 99.98% accuracy on the BraTS 2020 dataset.

Phase 1: Glioma Classification

The Efficient Visual Geometry Group CNN (EVGG-CNN) performs binary classification to determine glioma presence in MRI scans. The architecture includes 4 convolutional blocks with batch normalization and dropout.

Phase 2: Grade Detection

Modified Firefly Optimizer segments the tumor region. Shape and texture features are extracted and classified using SVM with RBF kernel to determine tumor grade (WHO Grade I-IV).

Key Features

  • Shape & texture feature extraction
  • SVM classifier for grade detection
